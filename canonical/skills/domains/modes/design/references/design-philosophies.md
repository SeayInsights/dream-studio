## I. Information Architecture School (01-04)
*Philosophy: "Data is not decoration — it's building material"*

### 01. Pentagram — Michael Bierut Style
**Philosophy:** Typography is language, grid is thought

**Characteristics:**
- Extreme color restraint (black/white + 1 brand color)
- Swiss grid system, modern interpretation
- Typography as primary visual language
- Strategic negative space (60%+ whitespace)

**Prompt DNA:**
```
Pentagram/Michael Bierut aesthetic:
- Extreme typographic hierarchy, Helvetica/Univers family
- Swiss grid with precise mathematical spacing
- Black/white + one accent color
- Information architecture as visual structure
- 60%+ whitespace ratio
- Data visualization as primary decoration
```

**Best for:** Corporate identity, B2B platforms, information-dense dashboards  
**Trade-off:** Conservative, won't surprise — prioritizes clarity over novelty  
**Reference work:** Hillary Clinton 2016 campaign identity

---

### 02. Stamen Design — Data Poetics
**Philosophy:** Make data into touchable landscapes

**Characteristics:**
- Cartographic thinking applied to information design
- Algorithm-generated organic forms
- Warm data visualization palette (terracotta, sage green, deep blues)
- Interactive layered systems

**Prompt DNA:**
```
Stamen Design aesthetic:
- Cartographic approach to data visualization
- Organic, algorithm-generated patterns
- Warm palette (terracotta #D4846A, sage green #8B9D83, deep blues #1F4E78)
- Layered information like topographic maps
- Hand-crafted feel despite digital precision
- Soft shadows and depth
```

**Best for:** Data dashboards, geospatial visualization, analytics platforms  
**Trade-off:** Requires real data to shine — weak with placeholder content  
**Reference work:** Stamen COVID-19 surge map

---

### 03. Information Architects — Content-First Principle
**Philosophy:** Design is not decoration — it's the architecture of content

**Characteristics:**
- Extreme content hierarchy clarity
- System fonts only (optimized for reading)
- Classic blue hyperlinks (#0000EE)
- Performance as aesthetics

**Prompt DNA:**
```
Information Architects philosophy:
- Content-first hierarchy, zero decorative elements
- System fonts only (SF Pro/Roboto/Inter)
- Classic blue hyperlinks (#0000EE)
- Reading-optimized line length (66 characters)
- Progressive disclosure of depth
- Text-heavy, fast-loading design
```

**Best for:** Editorial platforms, writing tools, documentation sites  
**Trade-off:** Deliberately "unsexy" — prioritizes function over visual appeal  
**Reference work:** iA Writer app

---

### 04. Fathom Information Design — Scientific Narrative
**Philosophy:** Every pixel must carry information

**Characteristics:**
- Scientific journal rigor + design elegance
- Precise quantitative data visualization
- Cool professional palette (grays, navy blue)
- Integrated citation and footnote systems

**Prompt DNA:**
```
Fathom Information Design:
- Scientific journal aesthetic meets modern design
- Precise data visualization (charts, timelines, scatter plots)
- Neutral scheme (grays #767676, navy #1F3A68, one highlight color)
- Footnote/citation design integrated into layout
- Clean sans-serif (GT America/Graphik)
- Information density without clutter
```

**Best for:** Annual reports, research publications, scientific infographics  
**Trade-off:** Requires accurate data — can't fake it with approximations  
**Reference work:** Bill & Melinda Gates Foundation annual reports

## II. Motion Poetics School (05-08)
*Philosophy: "Technology itself is flowing poetry"*

### 05. Locomotive — Scroll Narrative Master
**Philosophy:** Scrolling is not browsing — it's a journey

**Characteristics:**
- Silky parallax scrolling
- Cinematic scene composition
- Bold spatial whitespace
- Precise choreography of dynamic elements

**Prompt DNA:**
```
Locomotive scroll narrative:
- Film-like scene composition with parallax depth
- Generous vertical spacing between sections
- Bold typography emerging from darkness
- Smooth motion blur effects
- Dark mode (near-black #0A0A0A backgrounds)
- Strategic glowing accents
- Hero sections 100vh tall
```

**Best for:** Marketing sites, campaign launches, storytelling experiences  
**Trade-off:** Performance cost, not suitable for content-heavy sites  
**Reference work:** Lusion.co website

---

### 06. Active Theory — WebGL Poet
**Philosophy:** Making technology visible makes it understandable

**Characteristics:**
- 3D particle systems as core elements
- Real-time rendered data visualization
- Mouse-interactive world building
- Neon + deep space color schemes

**Prompt DNA:**
```
Active Theory WebGL aesthetic:
- Particle systems representing data flow
- 3D visualization in depth space
- Neon gradients (cyan #00F0FF, magenta #FF00E5, electric blue #0080FF) on dark
- Mouse-reactive environment
- Depth of field and bokeh effects
- Floating UI with glassmorphism
```

**Best for:** Tech demos, 3D product showcases, immersive experiences  
**Trade-off:** Heavy WebGL dependency, accessibility challenges  
**Reference work:** NASA Prospect (Active Theory)

---

### 07. Field.io — Algorithmic Aesthetics
**Philosophy:** Code is the designer

**Characteristics:**
- Generative art systems
- Dynamic graphics (different every visit)
- Intelligent composition of abstract geometry
- Balance between technical and artistic

**Prompt DNA:**
```
Field.io generative design:
- Algorithm-driven patterns, never the same twice
- Abstract geometric composition
- Monochrome or limited palette
- Real-time generation with code
- Mathematical beauty revealed
- Structured chaos
```

**Best for:** Creative tech brands, generative art projects, unique experiences  
**Trade-off:** Harder to control exact output, requires generative expertise  
**Reference work:** Field.io studio site

---

### 08. Resn — Motion Design Excellence
**Philosophy:** Every transition tells a story

**Characteristics:**
- Physics-based motion
- Seamless page transitions
- Bold color blocking
- Meticulous animation timing

**Prompt DNA:**
```
Resn motion design:
- Physics-based easing (spring, inertia)
- Bold color blocks in motion
- Seamless scene transitions
- Layered depth with parallax
- Considered timing (nothing instant)

```

**Best for:** Interactive experiences, product launches, creative portfolios  
**Trade-off:** Animation-heavy, needs performance optimization  
**Reference work:** Resn studio projects

## III. Experimental & Conceptual (09-12)

### 09. Experimental Jetset — Typographic Experimentation
**Philosophy:** Type is image, image is type

**Characteristics:**
- Typographic composition as primary visual
- Grid-breaking intentional asymmetry
- Bold conceptual statements
- Monochrome or highly restrained color

**Prompt DNA:**
```
Experimental Jetset:
- Typography as primary visual element
- Intentional grid-breaking (but still structured)
- Bold scale contrasts
- Conceptual messaging
- Black/white with one accent
- Cultural/intellectual references
```

**Best for:** Cultural institutions, galleries, avant-garde brands  
**Trade-off:** Polarizing — some see genius, others see pretentious  
**Reference work:** Stedelijk Museum identity

---

### 10. Müller-Brockmann — Swiss Perfection
**Philosophy:** Form follows function, grid follows mathematics

**Characteristics:**
- Mathematical grid precision
- Objective, universal communication
- Sans-serif only (Akzidenz-Grotesk, Helvetica)
- Ultra-clean, functional

**Prompt DNA:**
```
Müller-Brockmann Swiss style:
- Mathematical grid (8-column, 12-column)
- Objective sans-serif typography
- Primary colors or monochrome
- Absolute alignment discipline
- Every element justified to grid
- Timeless, not trendy
```

**Best for:** Corporate systems, wayfinding, technical documentation  
**Trade-off:** Can feel cold or sterile if misapplied  
**Reference work:** Zürich Tonhalle concert posters

---

### 11. Build — Contemporary Minimalism
**Philosophy:** Refined simplicity with subtle details

**Characteristics:**
- Clean contemporary aesthetic
- Subtle refinements (kerning, spacing, micro-interactions)
- Confident whitespace
- Modern serif + sans pairings

**Prompt DNA:**
```
Build contemporary minimal:
- Clean sans-serif (Founders Grotesk, Söhne)
- Subtle details (letter-spacing, optical sizing)
- Generous whitespace (40-50%)
- Refined color palette (not stark black/white)
- Micro-interactions that delight
- Premium feel without ostentation
```

**Best for:** Modern brands, design studios, premium products  
**Trade-off:** Risk of looking generic if not executed perfectly  
**Reference work:** Build portfolio site

**Philosophy:** Design should provoke, not just communicate

**Characteristics:**
- Provocative conceptual thinking
- Handmade elements mixed with digital
- Unexpected materials and executions
- Humor and humanity

**Prompt DNA:**

Sagmeister & Walsh experimental:
- Bold conceptual statements
- Handmade/tactile elements
- Unexpected materials (bananas, bodies, inflatables)
- Vibrant colors without restraint
- Humor and surprise

## V. Eastern & Material (17-20)

### 17. Takram — Design Engineering
**Philosophy:** Form and function are inseparable

**Characteristics:**
- Thoughtful restraint
- Form follows insight (not just function)
- Material honesty
- Interdisciplinary thinking

**Prompt DNA:**
```
Takram design engineering:
- Form + function unity
- Restrained palette (grays, whites, one accent)
- Material-conscious design
- Interdisciplinary approach (design + engineering)
- Thoughtful details, not decoration
- Honest communication
```

**Best for:** Product design, innovation projects, design systems  
**Trade-off:** Requires deep product understanding  
**Reference work:** Takram project case studies

**Characteristics:**
- Vast whitespace (70%+ empty)

Kenya Hara Eastern minimalism:

**Reference work:** MUJI design direction, Kenya Hara books

## Selection Guide

**For vague briefs, ask yourself:**

1. **What's the project type?**
   - Data-heavy → Pentagram, Stamen, Fathom, Müller-Brockmann
   - Marketing/storytelling → Locomotive, Resn, Sagmeister & Walsh
   - Tech/3D → Active Theory, Field.io, Ash Thorp
   - Cultural/editorial → Experimental Jetset, Kenya Hara, Irma Boom

2. **What's the risk tolerance?**
   - Conservative → Pentagram, Information Architects, Build
   - Moderate → Fathom, Takram, Müller-Brockmann
   - Bold → Sagmeister & Walsh, Experimental Jetset, Active Theory

3. **What's the execution path?**
   - HTML-first → Pentagram, Information Architects, Locomotive
   - AI generation → Field.io, Ash Thorp, Territory Studio
   - p5.js/generative → Zach Lieberman, Raven Kwok
   - Print-first → Müller-Brockmann, Kenya Hara, Irma Boom

**Recommend 3 that maximize differentiation** — don't pick three from the same category. Good combo example: Pentagram (safe info architecture) + Field.io (bold generative) + Kenya Hara (contemplative minimal).
