# Testing References

## End-to-End Testing

### awesome-playwright (1.4k stars)
**URL**: https://github.com/mxschmitt/awesome-playwright

Curated Playwright resources including:
- Page Object Model patterns
- Locator strategies (data-testid preferred over CSS/XPath)
- Parallel execution (2.3x faster than Cypress)
- Visual regression and accessibility testing
- CI/CD integration patterns

**Why this matters**: Playwright is the modern E2E testing standard with superior speed, reliability, and developer experience. This repo provides patterns for maintainable test suites.
