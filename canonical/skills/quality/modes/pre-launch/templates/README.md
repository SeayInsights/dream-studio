# Pre-Launch Legal & Compliance Document Templates

Templates for documents required by pre-launch rules.

| Template | Rule | Description |
|----------|------|-------------|
| `terms-of-service.md` | pl-001 | Terms of Service skeleton |
| `privacy-policy.md` | pl-002 | Privacy Policy skeleton (GDPR-aligned) |
| `accessibility-statement.md` | pl-004 | WCAG 2.1 conformance statement |
| `gdpr-dpa.md` | pl-005 | GDPR Data Processing Agreement framework |
| `deployment-runbook.md` | pl-011 | Deployment runbook template |
| `rollback-procedure.md` | pl-012 | Rollback procedure template |
| `privacy-impact-assessment.md` | pl-016 | Privacy Impact Assessment (DPIA) template |

All templates are starting points requiring legal review before use in production.
Fill in the bracketed placeholders with project-specific information.
