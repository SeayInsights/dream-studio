# dream-studio

An opinionated Claude Code plugin: two engines (Build Pipeline + Improvement Loop), 18 skills, agent personas, and a status bar — portable across projects.

## Install

```bash
claude plugins install dream-studio
```

Or install from source:

```bash
claude plugins install file://path/to/dream-studio
```

## Status

**0.1.0 — scaffolding.** Manifest only; skills, agents, and hooks ship in subsequent phases. See [CHANGELOG.md](CHANGELOG.md).
