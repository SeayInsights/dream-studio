## Implementation Strategy

### Dual-Mode Design

The implementation uses a **try-import pattern** with two modes:

#### 1. Full-Featured Mode (reportlab available)

```python
try:
    from reportlab.lib import colors
    from reportlab.lib.pagesizes import letter
    from reportlab.platypus import SimpleDocTemplate, Paragraph, Table
    HAS_REPORTLAB = True
except ImportError:
    HAS_REPORTLAB = False
```

**Features when reportlab is installed:**
- Professional multi-page PDFs
- Styled tables with alternating row colors
- Custom fonts and colors
- Chart image embedding
- Automatic page breaks
- Branded headers and footers
- Page numbers

#### 2. Text-Only Fallback (no reportlab)

**Features when reportlab is NOT installed:**
- Plain text export in .pdf file
- All metrics and data preserved
- Chart placeholders (text-only)
- Clear installation instructions
- No runtime errors

### Key Design Decisions

1. **Graceful Degradation**: Never fail hard - always produce output
2. **Clear Feedback**: Print warnings when falling back to text mode
3. **Production Ready**: Full error handling and validation
4. **Type Safety**: Type hints throughout (with `Any` for optional dependencies)
5. **Windows Compatibility**: No Unicode emojis in console output (cp1252 encoding issues)

## PDF Output Features

### Page Layout
- **Size**: Letter (8.5" × 11")
- **Margins**: 0.75" left/right, 1" top/bottom
- **Automatic page breaks**: When content exceeds page height

### Styling
- **Title**: 24pt, centered, dark gray (#1a1a1a)
- **Headings**: 16pt, left-aligned, blue-gray (#2c3e50)
- **Body text**: 10pt Helvetica
- **Tables**: 
  - Header row: Dark background (#2c3e50), white text

- **Footer right**: "Page N"

## Integration Example

    report_type="summary",

## Error Handling

The implementation handles:

1. **Missing reportlab**: Falls back to text-only mode with clear warning
2. **Invalid report data**: Returns `(False, error_message)`
3. **Missing required keys**: Validates metadata and sections
4. **File write errors**: Catches and reports permission issues
5. **Chart images not found**: Uses placeholder text instead
6. **Encoding issues**: No Unicode emojis (Windows cp1252 compatibility)
