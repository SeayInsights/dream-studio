# meta/

Per-session artifacts written by dream-studio hooks.

## lessons/

Committed lessons captured by `dream-studio:learn`. Each file documents a
non-obvious pattern or mistake from a real build session — kept in the repo
so the knowledge survives across machines and contributors.

File naming: `YYYY-MM-DD-<slug>.md`

## draft-lessons/

Unreviewed lesson drafts auto-generated by the `on-context-threshold` hook
when a session hits the handoff threshold. Review and either promote to
`lessons/` or delete.
