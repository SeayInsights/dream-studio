# Design System Library

Curated collection of 5 design systems for the design mode. Each system provides a complete visual language, component palette, and styling guidelines for different creative contexts.

## The 5 Systems

### 1. **tech-minimal**
Clean, sans-serif-first aesthetic with generous whitespace and subtle gradients. Emphasizes clarity and information hierarchy.

**Use when:** Building tech products, SaaS dashboards, developer-facing interfaces, data visualizations.

**Key traits:** Monospace accents, neutral palette (grays + primary), micro-interactions, high contrast.

---

### 2. **editorial-modern**
Bold typography, serif + sans-serif pairing, generous margins and modular layouts. Magazine-influenced visual system.

**Use when:** Content platforms, publishing tools, portfolio sites, storytelling experiences.

**Key traits:** Large headlines, sidebar grids, varied text scales, editorial photography integration.

---

### 3. **brutalist-bold**
Unconventional layouts, raw typography, maximalist visual density, industrial color palette. Rejects polish in favor of honest structure.

**Use when:** Experimental interfaces, art direction, statement-making designs, anti-design contexts.

**Key traits:** Heavy borders, monospace-dominant, offset grids, limited colors (black/white/accent), visible structure.

---

### 4. **playful-rounded**
Organic curves, vibrant multi-hue palette, custom emoji/icon system, and child-friendly sans-serif. Conversational and approachable.

**Use when:** Consumer apps, gaming interfaces, education tools, wellness platforms, community spaces.

**Key traits:** Rounded corners (24-32px), colorful gradients, animated transitions, playful iconography.

---

### 5. **executive-clean**
Sophisticated neutrals, serif headlines, elegant spacing, premium materials (glass morphism, subtle shadows). Corporate yet contemporary.

**Use when:** Finance dashboards, executive reports, luxury brands, professional services, B2B platforms.

**Key traits:** Deep navy/charcoal, premium typography, glass morphism, refined animations, trust-signaling.

---

## How to Use

From the design mode (`dream-studio:domains design`), reference a system by name:

```
Use the tech-minimal system for this dashboard layout.
Apply playful-rounded styling to the onboarding flow.
```

Each system includes:
- Color palettes
- Typography scales
- Component patterns
- Layout grids
- Animation language
- Code snippets

Systems are documented in individual subdirectories (e.g., `tech-minimal/`, `editorial-modern/`).
